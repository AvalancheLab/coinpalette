CoinPalette

A simple API for cryptocurrency logo colors.

Request

http://localhost:3000/api/list

Response

API

Base URL: https://www.coinpalette.com/api

    • GET

      /coinlist

      Fetch a list of all coin names.

      GET

      /colors?{coins}

      Fetch a list of colors based on coin parameter. Omit parameters to fetch every coin color.

    Built with Node-Vibrant and CoinGecko's API
